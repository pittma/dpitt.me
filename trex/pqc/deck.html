What’s Wrong with Classical Cryptography?

A quantum computer can trivially find the order \(r\) of a randomly chosen \(1 < a < n\):

\[ a^r \equiv 1 \mod n \]

We know that \(n\) divides \(a^r - 1\), so \(n\) also divides (via the difference of squares):

\[ (a^{r/2} - 1)(a^{r/2} + 1) \]

So we compute \(\gcd(a^{r/2} \pm 1, n)\). If the result is nontrivial, which it is very likely to be, we have a factor of \(n\) and the private key can be cracked.
